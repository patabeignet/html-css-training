HTML 5 Essentials Cheat Sheet

September 2023 - GRS - Worldline
Mozilla documentation: awl.li/htmldocs
Golden rule: HTML is for semantics, CSS is for style

Base elements

  • html
    • head
      • title
      • meta
      • link
    • body
    • script
div
generic block element
span
generic inline element

Sectioning content

section
generic section of an app/document
main
dominant content of a document
article
an independent part of the document that can be extracted without losing its meaning
aside
content tangentially related to the content around
nav
section with navigation links
search
section with search features: search box, search params, filters...
header
group of introductory or navigational aids.
footer
footer for its nearest ancestor sectioning content
dialog
for pop-ins / modal windows

Text hierarchy

h1, h2, h3, h4, h5, h6
headings (respect hierarchy !)
p
paragraph block
small
smaller importance, by default has a lower font size
strong
high importance, by default in bold
em
put emphasis on text, by default in italic
del
indicate a deleted content, by default with a line through
mark
highlight text, by default in yellow

Text classification

a
link, use it with href attribute
abbr
abbreviation, use it with title attribute
time
used with datetime attribute to refer to a date and time
address
used to refer to a postal address
q, blockquote
quotes, either inline or in a block, by default surrounded with quotes
pre
preformatted text, preserve whitespaces, by default in 
monospace
code
snippet of code, by default in monospace

Lists

ul
unordered list
ol
ordered list
li
list item
dl
list of terms/definitions. Terms are in dt followed directly by definitions in dd
menu
list of menu actions used for navigation or application controls
datalist
list of autocomplete suggestions in option children to link with input

Tables

table
use it only for data tables !
thead, tbody, tfoot
header,
body and
footer of a table
tr
table row
th
column heading in a tr in a thead
td
data cell heading in a tr in a tbody

Forms

  • form
  • search: contains a search form
  • fieldset: group a list of inputs
  • legend: caption of a fielset
  • label: use with for attribute
  • button
  • textarea
  • input
    • types: text, number, range, color, search, tel, email, url, datetime, datetime-local, date, month, week, time, checkbox, range
  • select
    • option, optgroup
  • output: a computed result

Media and other content

img
if you need several sources for an image, use picture and source tags
audio, video
Use either the src attribute or source children elements.
svg
inline vectorial images
canvas
used for 2D/3D drawing/scenes
figure
wrap around a media or table to link it with a caption inside a figcaption tag
progress, meter
progress bar and scalar value output
